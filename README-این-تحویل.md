## خلاصه‌ی سه فیکس

1. **کد تخفیف:** تعویض نوع جلسه (حضوری/آنلاین) تخفیف واقعی را پاک می‌کرد ولی
   پیام «کد اعمال شد» می‌ماند — مراجع فکر می‌کرد تخفیف فعال است، در درگاه
   قیمت کامل حساب می‌شد. رفع: `key={mode}` روی `DiscountCodeField` (دو جا).
2. **کانال پروتکل:** جلسه‌ی زیر یک پروتکل، به‌جای فقط کانال انتخابی دکتر
   (مثلا واتساپ)، همه‌ی کانال‌های فعال دکتر (واتساپ + بله + ...) را نشان
   می‌داد. رفع: `SessionCard` حالا `pkgChannel` می‌گیرد و فیلتر می‌کند.
3. **کانال جلسه‌ی مستقل:** جلسات بدون پروتکل اصلا کانال مشخصی نداشتند. رفع:
   چوزر کانال قبل از «انتخاب زمان جلسه» + ستون تازه‌ی `meet_channel`.

`npx next build` روی هر سه فیکس → exit 0. اسکن رقم فارسی/اعراب: صفر.
